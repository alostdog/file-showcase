# 蓝标数字广告三产品对比总览

## 📊 执行摘要

本报告汇总蓝色光标数字广告业务旗下三个核心产品的深度调研分析，为营销方案决策提供参考。

| 产品 | 核心定位 | 主要市场 | 战略产品 | 文档 |
|------|----------|----------|----------|------|
| **亿动 (Madhouse)** | 出海营销全链路服务 | 海外 (美日韩/东南亚) | 亿帆出海 SaaS | [查看详细](./调研报告 - 蓝标 - 上海亿动_Madhouse.md) |
| **蓝瀚互动 (BlueVision)** | 程序化广告与智能营销 | 国内 | DSP/DMP/Data+ | [查看详细](./调研报告 - 蓝标 - 蓝翰互动.md) |
| **Madhouse 海外营销** | 游戏/品牌/跨境电商出海 | 海外全区域 | 亿帆出海 + 本地化团队 | [查看详细](./调研报告 - 蓝标-Madhouse 海外营销.md) |

## 🎯 三产品关系说明

```
蓝色光标集团 (BlueFocus)
│
├── 数字广告业务板块
│   │
│   ├── 亿动广告 (Madhouse/上海亿动) ← 公司主体
│   │   └── 海外营销业务 ← 业务线
│   │       ├── 游戏出海方案
│   │       ├── 品牌出海方案
│   │       └── 跨境电商方案
│   │
│   ├── 蓝瀚互动 (BlueVision) ← 程序化广告子品牌
│   │   └── 专注国内市场 DSP/DMP 投放
│   │
│   ├── 蓝标传媒 (BlueMedia)
│   ├── 多盟 (Domob)
│   ├── 博杰传媒 (Bojie Media)
│   └── 其他子品牌...
│
└── 国际业务板块
    └── 蓝标国际 (BlueFocus International)
        └── 与亿动协同服务海外客户
```

**关键说明**：
- **亿动 (Madhouse)** 和 **Madhouse 海外营销** 是同一主体的不同视角文档
- **蓝瀚互动 (BlueVision)** 是独立的程序化广告子品牌，专注国内市场
- 三产品共享集团技术、数据、资源，可协同服务客户

## 📋 核心对比矩阵

### 一、定位与市场

| 维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|------|-----------------|----------------------|
| **核心定位** | 出海营销全链路服务 | 程序化广告与智能营销 |
| **主要市场** | 海外 (美日韩/东南亚/中东) | 中国大陆 |
| **目标客户** | 游戏厂商、品牌出海、跨境电商 | 国内大型企业、500 强、互联网 |
| **服务行业** | 游戏、消费品牌、电商 | 游戏、汽车、互联网、电商、金融等八大行业 |
| **品牌口号** | In Mad We Trust | - |

### 二、产品与技术

| 维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|------|-----------------|----------------------|
| **战略产品** | 亿帆出海 SaaS | DSP/DMP/Data+ |
| **技术能力** | 海外营销 SaaS、数据分析 | 程序化投放、用户画像、AI 优化 |
| **媒体资源** | Google、FB、TikTok、海外本地媒体 | 国内主流媒体、智能电视、跨屏 |
| **核心功能** | 营销管理、效果优化、资源对接 | 程序化购买、受众定向、实时竞价 |

### 三、服务与方案

| 维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|------|-----------------|----------------------|
| **核心方案** | 游戏出海、品牌出海、跨境电商 | 程序化投放、智能营销、跨屏广告 |
| **服务链条** | 市场洞察→创意→投放→转化→品牌 | 数据→定向→投放→优化→归因 |
| **本地化** | 59 个全球业务网络，本地团队 | 国内重点城市覆盖 |
| **交付模式** | 项目制/年框 + SaaS 订阅 | 服务费 + 媒体返点 |

### 四、商业模型

| 维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|------|-----------------|----------------------|
| **收入来源** | 服务费、SaaS 订阅、媒体返点、咨询 | 服务费、媒体返点、数据服务 |
| **客户预算** | 20-500 万/月 | 30-500 万/月 |
| **收费模式** | 项目制/年框 + SaaS | 服务费 + 返点 (5-15%) |
| **关键指标** | NDR>120%、ROI 提升>30%、NPS>50 | ROI 提升、续费率、跨屏覆盖率 |

### 五、竞争优势

| 维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|------|-----------------|----------------------|
| **核心优势** | 全链路 + 本地化 + SaaS | 技术 + 数据 + 全渠道 |
| **集团赋能** | 608 亿营收、59 个全球网络 | 608 亿营收、海量数据、AI 能力 |
| **竞争壁垒** | 早期出海经验、本地化团队 | 自有 DSP/DMP 技术栈、行业深耕 |
| **主要对手** | 飞书深诺、易点天下 | 阿里妈妈、腾讯广告、字节巨量 |

### 六、SWOT 对比

| 分析维度 | 亿动 (Madhouse) | 蓝瀚互动 (BlueVision) |
|----------|-----------------|----------------------|
| **优势 S** | 全链路服务、全球网络、SaaS 产品 | 技术产品矩阵、数据资源、全渠道 |
| **劣势 W** | 价格门槛高、人才依赖 | 品牌独立度低、技术同质化 |
| **机会 O** | 出海浪潮、新兴市场、AI 营销 | CTV 增长、AI 应用、第一方数据 |
| **威胁 T** | 隐私法规、平台政策、地缘政治 | 平台垄断、隐私法规、技术迭代 |
